# Linkedin-Network-Analysis

This project analyzes a LinkedIn-style graph of students and their connections to companies. It includes:
- **Cleaning** of raw graph data
- **Network analysis** (degrees, walks, paths)
- **Final visual report and tutorial**

##  Key Techniques
- Graph Theory (Degree, Walks, Pathfinding)
- NetworkX for graph analysis
- Matplotlib & Seaborn for visualization
